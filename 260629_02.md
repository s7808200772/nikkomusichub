# NikkoMusicHub 開發日誌 2026-06-29

## 本輪目標

完成使用者回饋的 Cloud / Pi UI 調整、新增雙端「版本更新 / Changelog」頁面、重新部署 Cloud Production、更新 Linear 並產出報告。

## 完成項目

### Cloud (Vercel)

- **修復 MQTT 預設帳密顯示**：SettingsClient mount 時會重新 fetch `/api/settings`，儲存成功後也會用回傳值更新 state，避免 placeholder 覆蓋實際值。
- **OTA / 音樂庫店點卡片 checkbox 排版**：checkbox 加上 `flexShrink:0`，文字區塊使用 `flex:1` 並單行省略，避免店點名稱被擠壓。
- **音樂庫 NAS WebDAV 設定**：新增 NAS URL、Remote Music Path、Username、Password 設定卡；載入清單後持久化到 `localStorage`，並顯示最後更新時間。
- **新增店點 store- 前綴寫死**：Store ID 會自動補上 `store-` 前綴，MQTT 帳密欄位提示改為「留空即使用預設值」。
- **總覽控制台中央控制台下移**：CommandsClient 已位於 DashboardClient（店點列表）下方。
- **版本更新頁面**：新增 `/changelog` 與 `/api/version`。

### Pi (Raspberry Pi)

- **MQTT 設定預設值與儲存不跳回**：`save_mqtt_settings` / `get_mqtt_settings` 改為直接讀寫 `nikko.env`，預設值為 `114.55.1.51:1883` / `admin` / `topup30%off`，並移除設定頁的 Topic Prefix / Command Secret 欄位。
- **WebDAV 設定簡化**：移除 Remote Name 欄位，Username 上移至原 Remote Name 位置；Remote Music Path 顯示改為 `\NikkoMusic` 形式，後端儲存時轉回 `nikko_nas:NikkoMusic`。
- **店家資訊簡化**：移除裝置 ID 與角色欄位，Store ID 固定 `store-` 前綴。
- **首頁儀表動畫/綠燈**：狀態卡片改為無閃爍 DOM 更新，綠燈加入 CSS 脈衝動畫。
- **播放控制台美化**：按鈕改為 lucide 風格 SVG 圖示。
- **側邊欄對齊與版權年份**：修改密碼 / Logout 上移至 nav 區域，新增「版本更新」入口，版權年份改為 2026。
- **版本更新頁面**：新增 `/version`、`/api/version` 與 OTA / rollback 按鈕。

### 安裝腳本

- `install.sh` 預設 MQTT 改為 `114.55.1.51:1883` / `admin` / `topup30%off`（TLS off）。
- 舊版 `broker.hivemq.com:8883` 會在 repair/install 時自動遷移。
- 初始化 Store ID 時自動加上 `store-` 前綴並寫入 SQLite。
