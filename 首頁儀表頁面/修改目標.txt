## 八、目前 UI 必須修掉的問題

請明確修掉以下問題：

1. 不要再有右側固定獨立狀態欄。
   圖1圖2右邊那條狀態欄請取消，改成圖3的主內容 Grid 右側卡片。

2. 不要出現頁面中間白色滾動條。
   主畫面不要被拆成左右兩個獨立 scroll container。
   整頁只允許 body/main 正常滾動；必要時只有 log terminal 內部可 scroll。

3. 不要讓 Toast 警告壓住右側卡片。
   預設密碼警告放在右上角，但要有 z-index 與位置控制，不要蓋住主要內容太多。
   或者放在 Header 下方做一條 warning banner，但高度要固定。

4. 播放清單不能再出現 [object Object]。
   必須正確顯示曲名。

5. 所有卡片要同一套設計語言。
   不要有些卡片很高、有些很矮、有些邊距不同。

6. 主內容左側不要貼著 sidebar。
   main content padding 至少 24px。

7. Header 不要遮住內容。
   如果 Header fixed，main 區要有 padding-top。

8. 整體請照圖3，不要自由發揮成新的樣子。

## 九、CSS / Tailwind 實作方向

如果使用 Tailwind，請大致照這些類型：

整體：
- body: bg-[#07111f] text-slate-100
- sidebar: w-[230px] bg-[#081426] border-r border-slate-700/40
- main: flex-1 bg-[#07111f] px-6 py-4 overflow-y-auto
- card: bg-[#111d31]/95 border border-slate-700/50 rounded-xl shadow-lg
- card title: text-cyan-400 font-semibold
- muted text: text-slate-400
- primary button: bg-blue-600 hover:bg-blue-500
- success button: bg-green-600 hover:bg-green-500
- danger button: bg-red-500 hover:bg-red-400
- neutral button: bg-slate-700/70 hover:bg-slate-600

Grid 建議：
- main dashboard container: grid grid-cols-12 gap-4
- 播放控制台：col-span-8
- 右側資訊卡區：col-span-4 grid grid-cols-2 gap-4 或 col-span-4 內部再 grid
- 音樂庫：col-span-6
- 播放清單：col-span-6
- 維護操作：col-span-12
- 系統 Log：col-span-6
- 操作紀錄 / 播放器操作紀錄：col-span-6，內部上下堆疊

如果畫面寬度不足：
- 1440px 以上照圖3
- 1024px 以下可改成單欄或兩欄，但桌機版必須優先照圖3
