## b. 本次變動項目

| 序號 | 項目 | 改變內容 | 如何驗收 | 預期效果 |
|---|---|---|---|---|
| 1 | Pi 音樂庫刪除按鈕改為批次刪除 | `app/templates/dashboard.html` 移除每列播放按鈕，將刪除改為標題列「刪除選取」，支援全選/取消全選與多檔批次刪除 | 進入 Pi 首頁「音樂庫」，勾選多個檔案後按「刪除選取」 | 減少誤刪、操作更直覺 |
| 2 | Pi shuffle/loop 勾選不再自動跳掉 | `app/services/mpv.py` 的 `get_status()` 改以持久化設定為 source of truth；前端寬限期延長至 3500ms，失敗時自動復原勾選 | 在 Pi 首頁勾選/取消「隨機播放」「循環播放」，等待 5 秒以上狀態保持 | 解決設定被 status poll 覆蓋的問題 |
| 3 | Pi 按鈕狀態與錯誤處理升級 | `app/static/app.js` 為 `apiGet`/`apiPost` 加上 timeout、`runAction` 支援逾時判斷、Toast 新增 success/error/warning 樣式、`setBusy` 可自訂文案 | 點擊任何維護按鈕，出現 loading 與結果 toast；中斷時顯示「操作逾時」 | 降低塑料感、強化回饋 |
| 4 | Cloud OTA 版本正確顯示 | `cloud-vercel/app/ota/OtaClient.js` 強化 `formatVersion`，支援 git object/string、version/git_version/commit/sha/ref 欄位，過濾 unknown/未知/none | 進入「店點管理 → OTA」，各店版本顯示 commit 前 7 碼或分支 | 不再顯示「未知」或「無法取得」 |
| 5 | Cloud 指令控制台輸出翻譯成人話 | `cloud-vercel/components/ResponseFormatter.js` 新增 sync/rescan/reboot/OTA/rollback/log/library_list/watchdog 等格式化器，通用 object key 也轉中文 | 在「店點指令控制台」執行任意指令，展開輸出為中文卡片 | 非技術人員也能看懂結果 |
| 6 | Cloud UI 質感升級 | `cloud-vercel/app/globals.css` 新增按鈕 progress bar、Toast 類色、focus ring、page fade-in、RWD 強化 | Cloud 各頁按鈕 hover/active/loading 有明確回饋，手機版不會爆版 | 整體操作更流暢 |
| 7 | Changelog 更新 | `cloud-vercel/app/changelog/ChangelogClient.js` 於 v2026.06.29 條目加入本輪 5 項重點 | 開啟「版本更新」頁面可看到本輪紀錄 | 版本紀錄同步 |
| 8 | Commit / Push / Vercel 部署 | `security-final` 已 push commit `8e01e8e`，Cloud Production 執行 `vercel --prod` 成功 | 開啟 `https://cloud-vercel-xi.vercel.app` 可看到新版 UI | 線上環境同步 |
