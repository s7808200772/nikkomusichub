# Nikko SoundNode｜日光音樂節點：120 項目完成狀態總覽

> 本文件依據目前 `security-final` branch 的實際程式碼與已部署狀態進行盤點。  
> 狀態分類：**已完成** / **部分完成** / **未開始**。  
> 「必要性」欄位代表「在目前正式營運情境下，是否建議補完或強化」。

---

## 說明與分類原則

- **已完成**：功能已實作且可在 UI / API / systemd 中實際運作。
- **部分完成**：核心邏輯或 API 已存在，但缺少 UI、缺少保護、缺少文件、或尚未達到可獨立營運的程度。
- **未開始**：程式碼或設定中找不到對應實作。
- **必要性**：
  - **必須**：P0 等級，直接影響門市播放或基本安全。
  - **建議**：P1 等級，可明顯提升穩定性、可維護性或使用者體驗。
  - **可選**：P2/P3 等級，規模化或進階場景再導入。
